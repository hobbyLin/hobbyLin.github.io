 Project Description

“株洲小巨蛋”

将打造全国最大的专业服饰展厅

小巨蛋采用无柱式钢结构构筑而成，中空高度达15米，演出场地1200平方米，后台区400平方米，并有专门的VIP 贵宾室、迎宾接待区近千平方米，容纳人数800人以上，是国内首屈一指的大型服饰发布会、订货会的专业场地，将承接国内外一线品牌的发布会、订货会等，并可用于承办国际时装周、国际模特大赛，国际设计师大赛等大型赛事和活动。

小巨蛋除了高标准的建筑、装修、灯光、音响等硬件设施外，还将引进国内一流的服务团队：广州SMG森明模特演艺、株洲新高度时尚模特等专业运营机构，为服饰品牌在小巨蛋举办展会提供专业服务。

小巨蛋外壳由LOW—E玻璃及彩色铝板搭配构接而成，其外立面璀璨夺目，契合服饰行业时尚潮流的特质，配以下方宽20米、长40米的银河天幕，打造芦淞新商圈的城市主立面，彰显城市地标，同时也将是国际服饰业界的地标。假以时日，各大品牌将以到“株洲小巨蛋”举办展会为事业之巅峰！
